
# gRPC-Managed Virtual Network Gateway

Built a lightweight cloud networking control-plane prototype written in **Rust** . It exposes a high-performance **gRPC API** to dynamically manage virtual VPN tunnels, manipulate routing tables, and inspect network packet structures asynchronously. 

Project was built to improve systems programming, high-concurrency network engineering, and production-level Rust architectures.

---

## Technical Stack

*   **Language:** Rust (Stable)
*   **Async Runtime:** Tokio
*   **RPC Framework:** Tonic (gRPC) / Prost (Protobuf Serialization)
*   **Database Interface:** SQLx (PostgreSQL)
